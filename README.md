1. 概述

   针对mybatis自动生成新增lombok插件，service插件（自动生成mapper对应的service），以及针对数据库UNSIGNED类型的字段，对应Java属性字段类型自动升级（避免插入字段值溢出）

2. 使用

   直接使用[base-mybatis-generator-plugin](https://github.com/liaomengge/base-mybatis-generator-plugin)插件即可
